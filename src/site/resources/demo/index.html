Demo

1. Download the stream-runner.jar.

2. Create a very simple process file (e.g. demo.xml): 

   <container>

      <stream id="example" class="stream.io.CsvStream" limit="10"
                 url="http://www.jwall.org/streams/demo.csv" />

      <process input="example">
         <PrintData />
      </process>

   </container>

3. Run the process: 

   # java -jar stream-runner.jar demo.xml
